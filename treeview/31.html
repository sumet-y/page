Network Address Translation (NAT) คืออะไร

Network Address Translation (NAT) เป็นเทคโนโลยีที่ใช้ในเครือข่ายคอมพิวเตอร์เพื่อแปลงที่อยู่ IP (IP address) ระหว่างเครือข่ายภายในและภายนอก เพื่อให้อุปกรณ์ในเครือข่ายภายในสามารถเชื่อมต่อกับอินเตอร์เน็ตหรือเครือข่ายภายนอกได้ โดยที่ไม่จำเป็นต้องมีที่อยู่ IP สาธารณะสำหรับแต่ละอุปกรณ์ภายในเครือข่ายภายใน.

NAT ทำงานโดยแปลงที่อยู่ IP และพอร์ต (port) ของแพ็กเก็ตข้อมูลเมื่อมีการส่งข้อมูลออกจากเครือข่ายภายในไปยังเครือข่ายภายนอกหรืออินเตอร์เน็ต โดยอุปกรณ์ NAT จะจดจำการส่งข้อมูลและตั้งเรื่องสำหรับการรับข้อมูลกลับมา ซึ่งทำให้เครือข่ายภายในสามารถใช้ที่อยู่ IP ภายในได้อย่างมีประสิทธิภาพ โดยไม่ต้องมีที่อยู่ IP สาธารณะสำหรับแต่ละอุปกรณ์

NAT มักถูกใช้ในอุปกรณ์เราเตอร์ (router) ที่เชื่อมต่อเครือข่ายภายในขององค์กรหรือบ้านเข้ากับอินเตอร์เน็ต โดยที่จะมีหน้าที่แปลงที่อยู่ IP ของอุปกรณ์ภายในเป็นที่อยู่ IP สาธารณะที่มอบให้โดยผู้ให้บริการอินเตอร์เน็ต ซึ่งช่วยเพิ่มความปลอดภัยและลดการใช้ที่อยู่ IP สาธารณะในการสื่อสารกับโลกภายนอก และช่วยเพิ่มประสิทธิภาพในการจัดการที่อยู่ IP ในเครือข่ายภายใน
