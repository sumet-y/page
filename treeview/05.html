Bandwidth คืออะไร

Bandwidth คือ ความเร็วในการส่งผ่านข้อมูล

Bandwidth การทดสอบ
- netsh wlan show interface
- nPerf

Bandwidth สาเหตุที่ทำให้ช้า
1.ฝั่งของผู้ใช้
- อุปกรณ์ที่มีสเปกสูงและเชื่อมต่อทางไร้สาย (WiFi หรือ 5G) มักมีประสิทธิภาพดีกว่า
- การใช้งานบราวเซอร์ที่มีการโหลดหน้าเว็บหลายหน้าพร้อมกัน
- การใช้งาน การดาวน์โหลดไฟล์ขนาดใหญ่ การสตรีมวิดีโอ
2.ฝั่งของระบบเครือข่าย
- ความเร็วที่ท่านใช้บริการ
- ปัญหาที่ Server ปลายทาง
- ปัญหาติดขัดที่ Firewall
- ความสามารถของอุปกรณ์เครือข่าย
- ความแรงของสัญญาณ และการรบกวนทางไฟฟ้า
